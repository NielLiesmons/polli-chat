# Polli

![Polli](assets/branding/readme-hero.png)

## Why Polli

- **Private-first spaces** — Personal, group, or community drives on their own domain; ACLs per file and folder; nothing leaves a space unless you allow it.
- **Chat-first** — Talk in the same place as files, tasks, and posts — without a cluttered dashboard.
- **Standards, not a walled garden** — [Chatmail](https://github.com/chatmail/core) / Delta Chat core for messaging, [MNS](https://github.com/mlkut/mns) for identity, [Cryptree](https://github.com/mlkut/cryptree) for files and state, email inboxes so anyone can reach a space from a normal mail client.
- **Cheap to run** — Small VPS, home box, or a provider you trust; no megascale cloud required.

## Current status

Android app with encrypted chat (groups, mail, channels), voice, attachments, and MNS sigils. Shared drive and full space inboxes are in progress — see [docs/spaces](./docs/spaces/).
